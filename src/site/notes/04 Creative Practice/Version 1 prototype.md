This was made as a game companion crossed with a visual novel for CIM417
It was intended to be used to make roleplaying smoother at the table.
No dice, no maths, no xp tracking.

it answered the question 
How can we engage players, pre-game to quickly teach them the setting and rules, in an entertaining way - leaving more time for connection during the IRL game?

Downtime actions could be solo roleplaying campaigns, or it could serve a video or animated comic.


The whole thing is useful, but skip to 5:30 for the prototype itself - This was made in protopie, and I could use a website builder (probably Wordpress) and then make it as a PWA, or use something like ren'py with some plugins (less flexible). Unity would also be feasible, Godot would be good too, but id need to learn it.
