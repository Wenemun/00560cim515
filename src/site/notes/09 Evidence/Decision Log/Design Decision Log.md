# Design Decision Log

## How to use this

Every time the project changes direction, scope, structure, language, or method, add an entry.

Use this especially when deciding between worldbuilding and framework-building.

---

## 2026-06-04 - Framework First, World as Testbed

### Decision

The Capstone will focus on developing a framework for worldtelling across multimodal interactive media. The speculative world will be used as the creative testbed rather than being the sole object of study.

### Reason

This keeps the project research-led while still allowing the creative practice to demonstrate and test the framework.

### Scope impact

Reduces the risk of the project becoming only worldbuilding. Keeps the exegesis focused on transferable design principles.
