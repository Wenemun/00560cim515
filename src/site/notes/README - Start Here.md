# Capstone Obsidian Vault

This vault is set up for a Capstone project focused on:

> **Multimodal Experience Design: A Practice-Led Framework for Interactive Media, Narrative Systems, and Transmedia Audience Engagement Platforms**

Core focus:

> How a speculative world can be experienced across interconnected media forms with different levels of participation and agency.

## How to use this vault

Start with [[Maps of Content/MOC - Capstone\|MOC - Capstone]].

Use:
- **02 Sources** for PDFs, articles, books, links, and raw source material.
- **03 Reading Notes** for notes on individual sources.
- **04 Concepts** for reusable ideas like [[03 Concepts/Worldbuilding - WorldTelling/Worldbuilding - WorldTelling\|Worldbuilding - WorldTelling]], [[03 Concepts/Agency and Participation/Agency\|Agency]], [[Transmedia\|Transmedia]], and [[03 Concepts/Experience Design/Experience Design\|Experience Design]].
- **05 Methods** for methodology and research design.
- **06 Framework** for the actual model, principles, patterns, terminology, and manifesto.
- **07 Creative Work** for the prototype or testbed.
- **08 Writing** for the exegesis and assessment drafts.
- **09 Evidence** for screenshots, process journal entries, design decisions, research data, and iteration evidence.

## Workflow

1. Drop raw material into [[00 Inbox\|00 Inbox]] or [[02 readings\|02 readings]].
2. Create a [[10 Templates/Template - Reading Note\|Reading Note]] for each source.
3. Extract key ideas into concept notes.
4. Link concept notes to methods, framework notes, and writing drafts.
5. Record major choices in [[09 Evidence/Decision Log/Design Decision Log\|Design Decision Log]].
